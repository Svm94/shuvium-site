# 🕉️ Shuvium - AI-Powered Spiritual Wellness Platform

Overview
Shuvium is a modern FaithTech platform designed to seamlessly merge ancient Vedic wisdom with state-of-the-art AI architecture. This repository contains the frontend client, which serves as the interactive gateway for users to experience personalized daily routines, dynamic AI mentoring (Achyuta-Partha interface), and a robust spiritual credit system.

🏗️ System Architecture
The application is built for high performance, scalability, and strict accessibility standards.

Frontend Framework: Next.js (App Router) / React

Styling: Tailwind CSS with custom responsive design tokens

Backend Ecosystem (API): Python FastAPI

Database & Authentication: Firebase / Firestore

Hosting: Vercel (CI/CD pipeline integrated)

🚀 Local Development Setup
To run this project locally:

Clone the repository:
git clone https://github.com/Svm94/shuvium-site.git

Install dependencies:
pnpm install

Start the development server:
pnpm dev

Open http://localhost:3000 with your browser to see the result.
